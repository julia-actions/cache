# Repository Instructions

These instructions apply to work performed by LLM coding agents in this repository.

## Repository Context

- This repository contains the `julia-actions/cache` GitHub Action.
- The action metadata lives in `action.yml`.
- The source code lives in `src/`, and the bundled action artifacts live in `dist/`.
- Run `npm ci` from the repository root before the first build in a fresh checkout, and whenever `package.json` or `package-lock.json` changes.
- If you add or remove npm dependencies, update the lockfile with `npm install`, `npm install <package>`, or `npm uninstall <package>` as appropriate.
- If a change updates the JavaScript action code, run `npm run build` from the repository root to rebuild `dist/` so it stays in sync with `src/`.

## Git Commits

- Add yourself as a co-author on every commit you make.

## Pull Requests

- Any pull request opened by an LLM coding agent must include a clear disclosure that it was generated by AI.
- Use a disclosure line equivalent to:
  `🤖 Generated by [name of AI tool].`
